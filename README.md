# 🏋️ RD Fitness Gym Website

A professional and responsive business website developed for **R.D Fitness Gym**. This project is designed to provide an engaging online presence with a modern UI, smooth animations, and mobile-friendly experience.

## 🌟 Features

- ✅ Responsive Design (Mobile, Tablet & Desktop)
- ✅ Modern UI/UX
- ✅ Hero Section
- ✅ About Gym Section
- ✅ Services Section
- ✅ Membership Plans
- ✅ Trainers Section
- ✅ Contact Section
- ✅ Google Maps Integration
- ✅ WhatsApp Contact Button
- ✅ Scroll Progress Bar
- ✅ Dark / Light Mode
- ✅ Loading Screen Animation
- ✅ Scroll Reveal Animations
- ✅ Back to Top Button
- ✅ SEO Friendly
- ✅ robots.txt
- ✅ sitemap.xml

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Responsive Design
- Google Fonts
- Font Awesome
